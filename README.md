# 特性

- **模块化**：基于SPI机制实现工作流节点的模块化处理，根据需求动态扩展、加载、替换或移除功能模块
- **灵活配置**：支持通过JSON配置文件或管理界面进行工作流的定制
- **基础规则节点**：支持常用的条件、自循环、多节点循环等基础规则节点，适应多样化的流程编排。
